### Objective

### Core Rules

- All work is conducted in a **private repository**
- Analysis proceeds under **pre-specified assumptions**
- No public release of intermediate results
- No retroactive parameter adjustment
- No introduction of new observables after Phase-1 closure

### Mandatory Components

1. **Data Ingestion & Pipeline Freeze**
   - Input datasets defined
   - Processing pipeline fixed before evaluation

2. **Decision Logging**
   - Every analytical choice recorded in `Decision_Log.md`
   - Includes:
     - rejected approaches
     - failed attempts
     - rationale for each decision

3. **Sensitivity & Robustness Testing**
   - Sensitivity Matrix construction
   - Dependence on assumptions explicitly mapped

4. **Null-Hypothesis & Bias Tests**
   - Control samples
   - Randomized or scrambled tests
   - Explicit systematics checks

- Complete analysis codebase
- Fully populated `Decision_Log.md`
- Results of all null and bias tests
- Internal summary of **procedure only** (not conclusions)

### Gate to Phase 3

Phase 2 is complete only when:

- The analysis pipeline is frozen
- All planned null tests are executed
- The Decision Log provides a complete audit trail
